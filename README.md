# Obra Integrada

Bem-vindo ao **Obra Integrada**.  
Uma plataforma web desenvolvida para modernizar o gerenciamento de obras na construção civil.

 

---

## Sobre o Projeto

O **Obra Integrada** foi criado para resolver problemas comuns no setor da construção civil, como:

- Uso de papel e anotações manuais  
- Falta de organização  
- Falhas na comunicação entre equipes  
- Desperdício de materiais  

A proposta é centralizar tudo em um único sistema simples, rápido e eficiente.

---

## Funcionalidades

O sistema conta com diversas funcionalidades:

- Login e autenticação de usuários  
- Cadastro de obras  
- Cadastro de cômodos  
- Cálculo automático de materiais  
- Geração de lista de compras  
- Resumo da obra  
- Indicação de profissionais  
- Fórum para interação entre usuários
