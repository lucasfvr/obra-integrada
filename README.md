# obra-integrada
Projeto acadêmico desenvolvido por estudantes do curso de Sistemas de Informação do UniFOA – Centro Universitário de Volta Redonda. A plataforma web para gestão de obras integra controle de etapas, calculadora de materiais, fornecedores e profissionais, permitindo acesso ao cliente e ao pedreiro para acompanhar e interagir durante todo o processo.
