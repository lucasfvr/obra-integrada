# 📱 Índice - Aplicativo Mobile (Canteiro de Obras)

Documentação do aplicativo mobile para operacional em canteiro de obras.

## Características Principais

### Funcionalidades Core
- Apontamento de horas (presença/ausência)
- Visualização de tarefas do dia
- Fotografias e documentação
- Sincronização offline-first
- Notificações em tempo real

### Recursos de Navegação
- Bottom Tab Navigation
- Deep linking
- Back button handling
- Gesture navigation

## 📱 Plataformas

### iOS
- Swift / SwiftUI
- Minimum iOS 13
- App Store deployment

### Android
- Kotlin / Jetpack Compose
- Minimum Android 9
- Play Store deployment
